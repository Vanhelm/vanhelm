Работа с Git
Создаем Репозиторий:
git config --global user.name "Максим"  задаём имя репозитория
git config --global user.email "mpohilchuk@gmail.com" задаем мыло репозитория
git config --list посмотреть настройки
cat ~/.gitconfig  посмотреть что в файле
git init  создание репозитория
git status статус git
git add workGit проиндексировать страницу перед камитом
git add . индексация всех файлов - не удалять файл
git commit "комментарий" закомить(сохранить)
git log - логи комитов
git show fc5a1f4bbda75d8021b22b936b09b4ce63eb2eee - посмотреть что было изененно в коммите//

git checkout work.txt - откатить изменения
git diff увидеть что изменили, но не индексировали
git cat увидеть, что в файле
git checkout хэш имя файла восстановить из хэша файл
git diff --staged увидеть проиндексированные изменения
git reset HEAD index.html исключить из камита
git log --oneline -1 красивый вид логов и 1 в списке
git commit --amend -m 'fkfk' переиминовать последний лог

git rm имя файла удалить файл из последнего комита
git commit --amend --no-edit изменить коммит после rm
git rm --cached путь к фалу - удалить файл из последнего камита, но не удалять из файлов
git help команда - справка

ВЕТКИ

git push -u origin master - отправить в удаленный репозиторий

Удаленная работа на 2-ух компах

git clone ссылка из gita сделать клон проекта
git checkout -b название ветки
git commit -a -m 'yfdpfybt' почему то так делают комит при отпраке в удаленный репозиторий
git push origin cments:comments добавить ветку в гит с норм названием
git push origin :cments удалить ветку удаленную 

git branch -m comments - переименовать(команда может также создавать удалять ветки)

git pull origin master получить файлы
git fetch origin comments получить ветку из удаленного  репозитория
git checkout -v comments(название ветки) origin/comments - создать ветку на основе удаленного репозитория
git branch --set-upstream-to=origin/comments - связать ветки локальную и удаленную 
git branch -vv посмотерть взаимосвящь веток очень подробно
git push and git pull работают после взаимосвязи
