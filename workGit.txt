Работа с Git
Создаем Репозиторий:
git config --global user.name "Максим"  задаём имя репозитория
git config --global user.email "mpohilchuk@gmail.com" задаем мыло репозитория
git config --list посмотреть настройки
cat ~/.gitconfig  посмотреть что в файле
git init  создание репозитория
git status статус git
git add workGit проиндексировать страницу перед камитом
git add . индексация всех файлов
git commit "text" закомить(сохранить)
git log - логи комитов
